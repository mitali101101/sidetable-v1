# sideTABLE

A simple book wishlist and shelf management tool.

## Features
- **Smart Shelf Logic:** Organize books by status (Reading, Wishlist, Finished).
- **Guest Access:** Fully functional "Anonymous" mode for quick testing without a Google account.
- **Robust Error Handling:** Implemented Error Boundaries to ensure the UI stays stable during network glitches.
- **Firebase Backend:** Real-time data synchronization with secure per-user database rules.

## Tech Stack
- **Frontend:** React 18, TypeScript, Vite
- **Styling:** Tailwind CSS, Lucide Icons
- **Backend:** Firebase (Auth, Firestore)
- **Deployment:** Vercel
